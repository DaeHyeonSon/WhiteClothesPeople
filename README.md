# 😜 주제

## 주선자가 보증하는 안전한 소개팅 서비스

- 주선자가 사용자 간의 소개팅을 주선하는 차별화된 소개팅 플랫폼
- 주선 과정의 불편함을 최소화하며 우소료(수수료)를 통해 동기 부여
- 주선자와 사용자의 관계 및 신원 보증 장치를 통해 안전한 소개팅 서비스 제공
